Бот для поиска фильма по коду🎬

Функциональный бот для заработка. Суть его работы в том, что человек получает информацию о фильме из TikTok или Shorts, после чего вводит соответствующий код в бот, который затем выдает название фильма.

— Рассылка текстом / картинкой
— Статистика (сколько человек воспользовалась ботом)
— Добавить канал
— Удалить канал из списка
— Удалить номер фильма

Больше сливов в канале https://t.me/+7xF6Jb3ka9A0ZDhi

Чтобы запустить установите python c офф сайта с галочкой add to path и библиотеку командой:
pip install aiogram==2.22.2

Укажите токен бота и ваш айди в файле main.py

Запуск:
py main.py
